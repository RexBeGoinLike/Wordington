
# Wordington Factorio

Word factory game written in Raylib C++



## Installation

Install the raylib executable file at https://www.raylib.com

Then on your terminal run:
```bash
  git clone https://github.com/RexBeGoinLike/Wordington.git
```
    
## Notes
## Node Types
Source Node
- The node that produces letters.
- Requires at least 1 logistic node to connect to a filter or receiver node.
- Input channels: 0.
- Output channels: 1 or more.

Logistic Node
- Conveyor Belts.
- Responsible for transporting text from one tile to another.
- Input channels: 1 or more (Depends on Type).
- Output channels: 1 or more (Depends on Type).

Filter Node
- Nodes that perform processing on a string.
- e.g. combine two letters/words/sentences, add spaces, punctuations, etc.
- Input channels: 1 or more (Depends on Type).
- Output channels: 1 or more (Depends on Type).

Receiver Node
- A node that accepts a required string.
- Randomly Generates said string using a library or fentgpt(?).
- Most likely generated by the game and not placed by a player.
- Input channels: 1 or more.
- Output channels: 0.
